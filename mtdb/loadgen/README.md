## Workload generator for long running applications

Runs on Ubuntu 15.10

### Prerequisite
```
$ sudo apt-get install -y maven gnuplot-nox
```

trustAnchors parameter must be non-empty -> [Help 1]
```

$ sudo update-ca-certificates -f
```
